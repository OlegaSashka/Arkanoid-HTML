# Arcanoid Vector Edition

Двумерный арканоид, разработанный на чистом **JavaScript** и **HTML5 Canvas** с использованием кастомного векторного физического движка. Проект построен на строгих принципах объектно-ориентированного программирования (ООП) и модульной архитектуры.

## Как запустить игру

Так как проект использует современную модульную структуру JavaScript (`type="module"`), обычный запуск файла `index.html` двойным щелчком мыши в проводнике будет заблокирован политикой безопасности браузеров (CORS). Для корректной работы игре необходим полноценный веб-сервер.

### Вариант 1. Запуск в один клик (Онлайн)

Вы можете запустить актуальную версию игры прямо в браузере, ничего не скачивая на компьютер. Игра автоматически собирается и публикуется через GitHub Pages:

👉 **[Играть в Arcanoid Vector Edition на GitHub Pages](https://твой-логин.github.io/название-репозитория/)**

### Вариант 2. Локальный запуск на компьютере (Разработка)

Самый быстрый и надежный способ поднять локальный сервер — использовать встроенный модуль **Python**. Он не требует установки тяжелого софта вроде Apache или OpenServer.

1.  **Проверка окружения:** Убедитесь, что у вас установлен Python. Откройте терминал (или командную строку Windows) и введите:
    ```bash
    python --version
    ```
    *Если команда не найдена, попробуйте `python3 --version`. Если Python не установлен, его можно скачать с официального сайта python.org.*
2.  **Переход в папку проекта:** Откройте терминал строго в корневой папке вашей игры (там, где лежит файл `index.html`).
    *Если вы работаете в VS Code, это можно сделать проще: нажмите `Ctrl + \`` (или меню `Terminal -> New Terminal` сверху).*
3.  **Запуск сервера:** Выполните команду для старта локального сервера:
    ```bash
    python -m http.server
    ```
    *Для систем macOS/Linux или если в Windows настроен вызов конкретно третьей версии, используйте:*
    ```bash
    python3 -m http.server
    ```
4.  **Открытие в браузере:** По умолчанию сервер развернется на порту `8000`. Кликните по ссылке ниже или вставьте её в адресную строку браузера:
    👉 **[http://localhost:8000](http://localhost:8000)**

---

## 🛠 Полезные команды при работе с сервером

* **Как остановить сервер:** Нажмите в терминали комбинацию клавиш `Ctrl + C` (на Mac — `Cmd + C`). Это безопасно отключит сервер.
* **Если порт 8000 уже занят:** Если у вас запущена другая программа или проект на этом порту, сервер выдаст ошибку. Вы можете принудительно запустить игру на любом другом свободном порту (например, `8080`), просто указав его номер в конце команды:
    ```bash
    python -m http.server 8080
    ```
    После этого игра будет доступна по адресу **[http://localhost:8080](http://localhost:8080)**.
